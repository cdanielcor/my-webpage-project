SQL

This is a hands on exercise I designed to practice the fundamentals of relational databases in MySQL. My goal was to create two connected tables and type various SQL queries to retrieve and analyze the data. I created a languages table and a products table to explore how programming languages relate to the applications built with them. The focus was on practicing CRUD operations and, most importantly, mastering different JOIN clauses to connect the data.



Java, Python, C++, C#, Ruby

This code (in different languages) is a basic project about a simple bank account and a couple of typical transactions. The best practice is to separate main and class in different files, but for simplifying purposes the code is shown together.

JAVA PYTHON C++ C# RUBY
class BankAccount {
private String owner;
private double balance;

public BankAccount(String owner, double balance) {
this.owner = owner;
this.balance = balance;
}

public void deposit(double amount) {
balance += amount;
System.out.println("Successful deposit.");
}

public void withdraw(double amount) {
if (balance >= amount) {
balance -= amount;
} else {
System.out.println("Insufficient funds.");
}
}

public void display() {
System.out.println(owner + "'s balance: $" + balance);
}

public static void main(String[] args) {
BankAccount acc = new BankAccount("Carlos", 1000);
acc.deposit(200);
acc.withdraw(1500);
acc.display();
}
}
